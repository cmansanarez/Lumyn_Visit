# LUMYN — Exhibition Landing Page
## Creative + Technical Direction for Claude Code

---

# PROJECT OVERVIEW

Create a landing page for **LUMYN: constellation of self**, an interactive installation by artist and creative technologist Cam Mansanarez (Noirmak).

This landing page is NOT the artwork itself.

It is:
- a threshold into the installation
- an exhibition information portal
- a conceptual framing space
- an atmospheric invitation

The page will be linked from:
- posters
- QR codes
- social promotion
- gallery marketing materials

The purpose is to:
- introduce visitors to the world of Lumyn
- provide exhibition details
- create anticipation
- direct visitors either:
  - to attend the installation
  - or optionally experience the web prototype

The page should feel:
- atmospheric
- spacious
- cinematic
- queer
- reflective
- celestial
- emotionally intimate
- contemporary gallery-grade

NOT:
- corporate
- startup-like
- overly UI-heavy
- cluttered

# 2. PROJECT DESCRIPTION

## Text:

*Lumyn: Constellation of Self* is an interactive installation of poetry, projection, and collective light created by artist and creative technologist Cam Mansanarez (Noirmak).

Visitors are invited to reflect on gentle prompts about identity, tenderness, memory, and inner brightness. Their responses are transformed in real time into luminous poetry, shifting gradients, and evolving constellation systems projected throughout the gallery space.

---

# 3. SECONDARY DESCRIPTION

## Text:

Part digital mirror and part collective archive, Lumyn explores queer self-recognition through generative technology and immersive visuals. Each participant contributes a fragment of light to a growing public constellation, creating an evolving environment shaped by vulnerability, reflection, and presence.

---

# 4. EXPERIENCE SECTION

## Heading:
What to Expect

## Text:

Visitors will engage with Lumyn through a private reflection station positioned within the gallery.

As responses are submitted, the surrounding projection environment transforms through color, stars, motion, and poetry. Over the course of the exhibition, the installation accumulates a collective constellation generated by the community itself.
